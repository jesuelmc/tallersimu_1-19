Instrucciones para introducir datos de simulación.
Paso 1
Selecione un número de salas que desee que esten disponibles para la simulación.
Paso 2
Arrastre el row switch para medir el número de jueces tecnicos deseados.

Paso 3
Seleccione los tipos de delitos disponibles para la simulacion.
Paso 4
Arrastre el row switch para medir el número de acusados con antecedentes.

Paso 5
Arrastre el row switch para medir el número de denuncias echas a la policia y al jusgado.

Instrucciones para la Animación de la simulación.
Paso 1
Preciona el boton "Empezar simulacion" para iniciar con la interaccion de simulacion.
Instrucciones para la verificacion de resultados.
Paso 1
Preciona el boton "Descargar" para iniciar una descarga de los resultados en una imagen completa.
